# Contributing to IBM Professional Activity

Thank you for your interest in contributing to this repository!

## Guidelines

### Adding New Blog Posts

1. **Create a new Markdown file** with a descriptive name (e.g., `Topic_Article_Title.md`)
2. **Use the following structure:**
   ```markdown
   # Article Title
   
   **Author:** Name  
   **Published on:** Date  
   **Web Address:** Original URL (if applicable)
   
   ---
   
   ## Content
   ```

3. **Reference images** from the `assets/` folder using relative paths:
   ```markdown
   ![Description](./assets/image-name.png)
   ```

4. **Update README.md** to list the new article

5. **Commit and push** with a clear commit message

### Adding Images

- Place all images in the `assets/` folder
- Use descriptive filenames (e.g., `article-title-diagram.png`)
- Supported formats: PNG, JPG, GIF, SVG
- Keep image sizes reasonable (< 2MB when possible)

### Commit Messages

Use clear, descriptive commit messages:
- `Add: Article title - brief description`
- `Update: Article title - changes made`
- `Fix: Issue description`

## Code of Conduct

- Be respectful and professional
- Focus on content quality
- Maintain accuracy of information
- Credit original sources and authors

## Questions?

For questions or suggestions, please open an issue or contact the repository maintainer.
