# GitHub Repository Setup Instructions

To connect this repository to GitHub:

## 1. Create Repository on GitHub
- Go to https://github.com/nolecram/ibm_professional_activity
- Create a new public repository (or use existing one)
- Do NOT initialize with README, .gitignore, or license

## 2. Add Remote and Push
Run these commands:

```bash
cd ~/ibm_professional_activity
git remote add origin https://github.com/nolecram/ibm_professional_activity.git
git branch -M main
git push -u origin main
```

## 3. Add Images to Assets Folder
Add these images to the `assets/` folder:
- `thoughts-on-cloud-banner.png`
- `openstack-folsom-architecture.png`  
- `marco-celon-photo.jpg`

Then commit:
```bash
git add assets/
git commit -m "Add blog post images"
git push
```

## 4. Verify
Check the repository at: https://github.com/nolecram/ibm_professional_activity
